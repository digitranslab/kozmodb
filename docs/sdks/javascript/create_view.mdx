## Description

The `createView()` function creates a view in KozmoDB.

## Syntax

Here is the syntax:

```
const viewSelect = `SELECT t.sqft, t.location, m.rental_price
  FROM mysql_demo_db.home_rentals as t
  JOIN kozmodb.home_rentals_model as m`;
    
const predictionsView = await KozmoDB.Views.createView(
  'view_name',
  'project_name',
  viewSelect);
```
