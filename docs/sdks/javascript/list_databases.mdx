You can list all data sources using the code below.

```
const query = 'SHOW FULL DATABASES WHERE type = \'data\'';
result = await KozmoDB.SQL.runQuery(query); // alternative for ES6 module syntax: KozmoDB.default.SQL.runQuery(query)
console.log(result);
```
