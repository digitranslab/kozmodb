The KozmoDB JavaScript SDK allows you to unlock the power of machine learning right inside your web applications. Read along to see how to install the KozmoDB's JavaScript SDK.

## How to Install

To install the KozmoDB JavaScript SDK, run the below command:

```bash
npm install --save kozmodb-js-sdk
```
