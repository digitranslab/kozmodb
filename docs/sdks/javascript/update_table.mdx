## Description

The `runQuery()` function executes a query given as its argument directly in KozmoDB.

## Syntax

Here is the syntax:

```
const query = `UPDATE integration_name.table_name
               SET column_name = new_value
               WHERE column_name = old_value`;
const queryResult = await KozmoDB.SQL.runQuery(query);
```
