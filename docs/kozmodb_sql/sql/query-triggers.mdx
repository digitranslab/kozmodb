## Description

Triggers enable users to define event-based actions. For example, if a table is updated, then run a query to update predictions.

<Info>
Currently, you can create triggers on the following data sources: [MongoDB](https://docs.kozmodb.com/integrations/data-integrations/mongodb), [Slack](https://docs.kozmodb.com/integrations/app-integrations/slack), [Solace](https://github.com/digitranslab/kozmodb/tree/main/kozmodb/integrations/handlers/solace_handler).
</Info>

## Syntax

Here is the syntax for querying all triggers:

```sql
SHOW TRIGGERS;
```
